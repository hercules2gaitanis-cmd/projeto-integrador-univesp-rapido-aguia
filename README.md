# projeto-integrador-univesp-rapido-aguia
Software de gestão de carga, cubagem e peso para a empresa Rápido Águia - Hyundai HR.
este software ajuda a Rápido Águia a não sobrecarregar o Hyundai HR (respeitando os 1.500 kg de limite) e a planejar melhor o espaço do baú.
# 🦅 Rápido Águia - Sistema de Gestão de Carga

## 📌 Visão Geral
Este projeto foi desenvolvido para o **Projeto Integrador em Computação I (PJI110)** da UNIVESP. O software auxilia a empresa **Rápido Águia** na otimização do carregamento de veículos Hyundai HR, garantindo segurança e eficiência logística.

## 🛠️ Requisitos Atendidos (Ementa Univesp)
* **Levantamento de Requisitos:** Identificação da necessidade de controle de peso (limite de 1.500kg) e cubagem (volumes de 10m³, 11m³ e 13.5m³).
* **Desenvolvimento Web:** Interface construída com HTML5 e CSS3.
* **Script Web:** Lógica de cálculo implementada em JavaScript.
* **Controle de Versão:** Gestão do código via GitHub.

## 🚀 Funcionalidades Principais
* **Cálculo de Capacidade Real:** O sistema cruza o volume disponível com o peso máximo permitido.
* **Resultado Autorizado:** Destaca o número final de unidades que podem ser transportadas com segurança.
* **Margem de Segurança:** Aplicação de 10% de folga no volume do baú para manuseio da carga.

## 📊 Regras de Negócio
- **Limite de Peso:** 1.500 kg (Carga Útil Líquida).
- **Fator de Folga:** 10% de desconto no volume bruto para acomodação real.
